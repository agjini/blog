# Hamon

Template for [Bar](https://github.com/Lurk/bar)

Icons by [Font Awesome](https://fontawesome.com/)

Used in: [barhamon.com](barhamon.com)

## Template config

### index_article_id

ID of an article to use on index page.

For example:

```yaml
index_article_id: '/main'
```

### index_tags

List of tags. Each tagâ€™s three most recent articles will be shown on the index page. The order of tags will be
preserved.

For example:

```yaml
index_tags:
  - life
  - books
  - music
```
